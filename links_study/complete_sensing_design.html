2. Ground Reference Strategy

Detailed file: ground_reference_diagram.html
Conclusion: Control board ground must reference DC− (the output power return), NOT AC−. A single star-point connection ties DC− to the control board AGND. All isolation devices (gate drivers, isolated amplifiers, isolated DC-DC) straddle the isolation barrier between the HV power domain and LV control domain.

Key Points

RuleImplementation
Control board ground referenceDC− (output power return)
AC− as reference?❌ Never — floats at 50 Hz, ±325 Vpk CM
Star pointSingle connection: DC− → AGND, near aux power supply return
AGND/DGND joinSingle point underneath the DSP
PE (earth)Connects to chassis only — never to control board ground
Isolation barrier≥8 mm creepage for 400 V (IEC 62368-1)

Production Notes

  1. Prototype build: Populate all DNP resistors (RDNP1, RDNP2) as 0 Ω. Populate LEM sensor first for validation. Swap to ACS37002/TMCS1133 later by removing RDNP1 and populating RDNP2.
  2. Production BOM variant 1 (LEM): RDNP1 = 0 Ω, RDNP2 = DNP. Populate LEM LA 55-P.
  3. Production BOM variant 2 (ACS): RDNP1 = DNP, RDNP2 = 0 Ω. Populate ACS37002.
  4. Single BOM: Include all components in the master BOM. Assembly house populates per variant instructions.
  5. Firmware: No changes needed between variants — analog front-end produces identical ADC input.
  6. Calibration: Store per-unit calibration constants in DSP flash for resistor divider tolerance (0.1% resistors → minimal calibration needed).
