LEM Sensor Replacement — Shared Feedback Connector Design

Designing a PCB that supports either the existing LEM LA 55-P Hall-effect sensor or a cheaper alternative (ACS37002, TMCS1133), with both sensors sharing the same feedback signal wire to the DSP connector. The switchover is handled via DNP (Do Not Populate) component options.

2. Output Signal Matching

The key challenge is that each sensor produces a different output voltage for the same current. We need to scale all sensor outputs to produce the same voltage at the DSP ADC input for a given current, so the DSP firmware doesn't need to change.

Target: Match LEM LA 55-P Output Characteristic

LEM LA 55-P with RM = 49.9 Ω:
• Sensitivity: 25 mV/A
• Output at 0 A: 0 V (bipolar supply, centered at 0 V)
• Output at +20 A: +0.500 V
• Output at −20 A: −0.500 V

But the DSP ADC needs 0–3.3 V input, so we need a level shift:
• Add 1.65 V offset to center the signal in the ADC range
• Target: Vout = 1.65 V + (25 mV/A × Isense)

At +20 A: Vout = 1.65 + 0.50 = 2.150 V (target for ALL sensors)
At −20 A: Vout = 1.65 − 0.50 = 1.150 V (target for ALL sensors)
At 0 A: Vout = 1.650 V (mid-scale, all sensors)

ACS37002 and TMCS1133 already output from a 1.65 V quiescent point (50% of 3.3 V supply). Their output swings are different from the LEM, so we need different gain/attenuation networks. The simplest approach is to design each sensor's output stage to produce the same 1.65 V ± (25 mV/A × I) signal.

3. Option A: DNP 0 Ω Series Resistors (Simple, Recommended for Production)

This is the simplest and most cost-effective approach. Each sensor has its own output path to the shared feedback connector. Only one sensor is populated; the other path is disabled by removing (DNP) the series 0 Ω resistor.

4. Scaling Network Calculations

ACS37002 (60 mV/A) → Target 25 mV/A

Required attenuation: 25/60 = 0.4167

Voltage divider: Rtop = 10 kΩ, Rbot = 7.15 kΩ (E96)
Gain = Rbot / (Rtop + Rbot) = 7.15k / 17.15k = 0.4169 ✓

Verification:
• ACS output at 20 A: 2.850 V (from 1.65 V quiescent, swing = +1.200 V)
• After divider: 1.65 + (1.200 × 0.4169) = 1.65 + 0.500 = 2.150 V ✓
• ACS output at −20 A: 0.450 V (swing = −1.200 V from 1.65 V)
• After divider: 1.65 + (−1.200 × 0.4169) = 1.65 − 0.500 = 1.150 V ✓

The quiescent point (1.65 V) is preserved because the divider is referenced to the same 1.65 V virtual ground.

TMCS1133A1 (40 mV/A) → Target 25 mV/A

Required attenuation: 25/40 = 0.625

Voltage divider: Rtop = 10 kΩ, Rbot = 16.9 kΩ (E96)
Gain = 16.9k / (10k + 16.9k) = 16.9 / 26.9 = 0.6283
Error from ideal: 0.6283 / 0.625 = 1.0053 → 0.53% error (acceptable)

For exact 0.625: Rbot = 16.67 kΩ → use 16.5 kΩ (E96) + 180 Ω trim, or 16.9 kΩ with calibration
Alternatively: Rtop = 12.1 kΩ, Rbot = 20.0 kΩ → Gain = 20/32.1 = 0.6231 (0.3% error)

Verification (using 16.9 kΩ):
• TMCS output at 20 A: 1.65 + 20 × 0.040 = 2.450 V (swing = +0.800 V)
• After divider: 1.65 + (0.800 × 0.6283) = 1.65 + 0.503 = 2.153 V (1.3 mV error = 0.06%) ✓
• TMCS output at −20 A: 1.65 − 0.800 = 0.850 V
• After divider: 1.65 − (0.800 × 0.6283) = 1.65 − 0.503 = 1.147 V (3 mV error) ✓

5. LEM Level Shift Circuit

The LEM LA 55-P outputs a bipolar signal (±0.5 V for ±20 A) centered at 0 V. The DSP ADC needs 0–3.3 V, so we need to add a 1.65 V DC offset.

8. DSP ADC Scaling — Firmware Reference

Regardless of which sensor is populated, the DSP always sees:
VADC = 1.65 V + (25 mV/A × Isense)

To recover current in firmware:
Isense = (VADC − 1.65) / 0.025

In ADC counts (12-bit, 3.3 V reference):
ADC_mid = 2048 (corresponds to 1.65 V = 0 A)
Isense = (ADC_code − 2048) × (3.3 / 4095) / 0.025
Isense = (ADC_code − 2048) × 0.0322 A/count

At ADC = 2489 (+20 A): I = (2489 − 2048) × 0.0322 = 14.2 × 0.0322 ≈ 20.0 A ✓
At ADC = 1607 (−20 A): I = (1607 − 2048) × 0.0322 = −441 × 0.0322 ≈ −20.0 A ✓
Key advantage of Option A: The DSP firmware is sensor-agnostic. It always applies the same scaling factor (25 mV/A equivalent), regardless of whether the physical sensor is a LEM LA 55-P, ACS37002, or TMCS1133. The analog front-end handles all the scaling and level shifting.
